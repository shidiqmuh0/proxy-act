# Proxy Act

Install dependecies

```bash
npm i fs axios path
```

Download proxy-act

```bash
wget -O proxies.js https://raw.githubusercontent.com/shidiqmuh0/proxy-act/main/proxies.js
```

Before run the code, put proxies on proxies.txt

Like this format

```bash
202.61.204.51:80
46.35.9.110:80
```

After that, you can run 

```bash
node proxies.js
```
Output will come out, on output.txt
